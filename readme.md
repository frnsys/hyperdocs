EXPERIMENTAL

p2p collaborative document editing, built on [`hypermerge`](https://github.com/automerge/hypermerge). Besides basic collaborative editing, you can also see peer names, cursors, and text selections.

# Usage

1. Clone this repo
2. `npm install`
3. `npm run compile`
4. `npm start`
